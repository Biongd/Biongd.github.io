potplayer使用教程

发表于 2020-07-08   |   更新于 2021-08-29 | 分类于 常用操作-excel.potplayer.快捷光标.markdown.下载.typora.windows.

对于新尝试的视频软件potplayer，无疑是带着一种发现新大陆的心情，但是在使用了一段时间后才意识到对于我这个门外汉来说再NB得东西也就当普通播放器使。正好在找看电视直播的软件的时候发现了这个使用这个软件的特殊姿势，学起来！

原文链接：https://post.smzdm.com/p/av7k34qm/

1. 界面优化

首先右键Potplayer窗口右键，选择“选项”，或者按下快捷键F5，进去选项设置界面；

1.1 无边框

进入”选项设置”之后，点开“基本”，找到“皮肤/配色”，单击打开，将下图中的两个选项进行勾选，最后点击右下角的“应用”；因为勾选之后，在播放视频就没有边框，并且菜单栏自动隐藏；

1.2 窗口优化

  1. 首次使用Potplayer的时候，点开视频会直接跳到的默认尺寸；如果视频和屏幕分辨率都是1080P，那么直接就是全屏，而我们需要将Potplayer启动窗口进行自定义调节；

    ps. 由于大多数电影、视频以及MV等，普遍采用是16：9的比例，那么个人建议，将窗口的尺寸设置为：480×272、720×405、1024×576、1280×720、1366×768、或1920×1080等等，根据自己电脑屏幕大小，选择其中一种即可；

    15.6寸笔记本设置的是：1280×720

  2. 找到“播放”选项，单击打开；点开“播放窗口尺寸”的下拉列表，同样选择其中的“用户自定义”，然后进行分辨率调节，最后点击右下角的“应用”；如下图，播放窗口尺寸可调为是1280×720；

1.3 皮肤设置

Potplayer播放窗口底下状态栏默认是不透明的，在播放的视频的时候，很容易遮挡字幕，我们可以调节透明度来改善；

  1. 进入”选项设置”之后，点开“基本”，找到“皮肤覆盖/放大”，单击打开；点开“方式”的下拉列表，选择其中的“使用Direct3D 9”，最后点击“应用”；
  2. 如果觉得透明度还不够的话，可以通过鼠标，滑动下图中的“不透明度”，来进一步调节，记得别忘了最后点击“应用”

1.4 皮肤安装

如果觉得Potplayer的默认皮肤不好看，想继续折腾的话，那么可以安装其他皮肤；

①. 首先在网上下载好Potplayer皮肤文件（随便找，有很多），然后回到”选项设置”，点开“基本”，选择“皮肤与配色”选项；找到其中的”打开皮肤文件夹“，点击打开；

2. 电视直播

查找potplayer直播源，直接拖动至软件的专辑栏内，可以新建专辑更改名称
